# ChartIqai

This project features a React/Next.js frontend and a FastAPI backend, designed for seamless deployment on Vercel.

## Structure
- `frontend/`: React/Next.js application.
- `api/`: FastAPI serverless functions.

## Local Development
1. Install frontend dependencies: `cd frontend && npm install`
2. Install API dependencies: `pip install -r requirements.txt`
3. Run the development server: `npm run dev`
